Vast Upload Tutorial

  1.  The first thing you need to do is download the file here:

  2.  Once downloaded, move the "inputFiles.py" into the same directory as your job file

  3.  In the command line, navigate to the directory where both are contained and then type "./inputFiles.py <job_file>" where <job_file> is the job file that will be parsed.

    1. The job_file will be parsed for valid jobs able to be offloaded to BOINC

    2. If that job is dependent on a file in order to be run, the inputFiles will create a new directory called "inputFiles" that will now contain a copy of the file that the job would need.

    3. At the end, all of the input files needed to run the offloaded jobs will be in the new inputFiles directory.

  4.  On the upload page, import the job file where it says "Job File" and import all of the input Files (in the directory "inputFiles") where it says "Input Files"

    1.  If there is more than one input file I recommend clicking the top file, scrolling all the way to the lowest file and shift-click on that file, which will highlight all of the input files and upload them all at once

  5. Click the "Upload" button.
